## 关于

jshook是对应用程序注入rhino/frida，你只需要会js就可以快速实现hook，并且支持java层和native层。

rhino包含了xposed的api，主要用于java层的hook；frida注入后js可以完全访问内存，支持java层和native层。

## 兼容

1. Xposed api 82
2. Android 5 - 13
